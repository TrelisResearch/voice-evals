# Voice Evals

Voice evaluation datasets and benchmarks for ASR models, built with [Trelis Studio](https://studio.trelis.com).

## Datasets

| Dataset | Tier | Languages | Source |
|---------|------|-----------|--------|
| `voice-evals-ai-technical` | Public + Semi-private | English | AI News, YouTube transcripts |
| `voice-evals-code-switching` | Public + Semi-private | FR, EN, DE, ES | Synthetic mixed-language |
| `Trelis-OOD` | Private | EN, EN-GA | ATC, code-switching, technical |

## Setup

```bash
cp .env.example .env
# Add TRELIS_API_KEY and HF_TOKEN
```
